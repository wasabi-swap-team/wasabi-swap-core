# Wasabi Factory

In-depth documentation on WasabiSwap is available at [https://wasabi-swap.gitbook.io/swapwasabi](https://wasabi-swap.gitbook.io/swapwasabi).

## Local Development

The following assumes the use of `node@>=10`.

### Install Dependencies

`yarn`

### Compile Contracts

`yarn compile`

### Run Tests

`yarn test`
